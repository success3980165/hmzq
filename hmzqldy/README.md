# 安装

1. 安装 `eslint`

```shell
npm install -g eslint eslint-config-standard eslint-plugin-standard eslint-plugin-promise
```

2. 安装编辑器 eslint 插件，推荐编辑器 `Sublime Text`、`Visual Studio Code`
